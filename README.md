# 🌍 World Happiness Report Analysis

## 📊 Project Overview
This project analyzes global happiness data using Python and data science techniques.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📁 Dataset

- Top happiest countries
- Correlation between GDP and happiness
- Impact of social support and life expectancy
